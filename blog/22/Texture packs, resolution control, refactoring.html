Texture packs, resolution control, refactoring

Published Jun 18, 2015

Over the past week, mrpimpunicorn has been sending some kick-ass pull requests to TrueCraft, including texture pack support, improved text rendering, optimizations, and more.

13 pull requests so far (one still open even now) is awesome. Great work, mrpimpunicorn. Updates include:

  • Texture packs
  • Resolution control in launcher
  • Input refactoring
  • Colors in chat and improved text rendering
  • XML documentation for the to-be modding API
