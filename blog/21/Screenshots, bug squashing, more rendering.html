Screenshots, bug squashing, more rendering

Published Jun 04, 2015

Press F2 on the client to take a screenshot! It'll end up in ~/.truecraft/screenshots/ or somewhere similar on other platforms. As you can tell, there's also support for rendering each variety of tree as well as ladders.

Support for screenshots just involved rendering to a render target instead of directly to the screen. Then that texture is drawn on the screen in 2D after the fact. We can save this texture to disk when the user requests a screenshot.

There was also some minor bug squashing with cross-thread access violations. This enables the client to connect to official Minecraft beta 1.7.3 servers without crashing after a while (the official server sends chunks over the network in a different manner from TrueCraft, which is why the bug was present when connecting to official servers but not TC servers).

Next up is support for rendering even more unusual blocks. These updates will probably be boring for a while as each one will consist of one or two new blocks that get custom block renderers. Hang in there, because once that's done we'll be working on lighting!
