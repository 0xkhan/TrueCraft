Chests, Gamocosm server hosting, nightlies

Published Jun 22, 2015

Lots of exciting TrueCraft news today. First, chests work! You can store items for later use on your TrueCraft servers now. Speaking of your servers, I've added support for TrueCraft to Gamocosm, an open source Realms alternative. If you want a TC server to play on, you can get one here pretty easily.

There are also now nightly builds of TrueCraft that you can download here. These are unstable and known to be broken on Windows. However, they are updated every time we update TrueCraft, within minutes, thanks to blha303 hosting the new build server.

In other news, mrpimpunicorn continues his streak of pull requests with support for rendering slabs and cobwebs in the client:

I took this screenshot using the official Mojang assets. Don't worry about the FPS, I'm stuck on a crappy laptop this week. I don't have any screenshots of this, but I also personally implemented support for rendering crops (wheat).

Manio143 and Mitch528 also added support for the query protocol to TrueCraft, so you can get detailed info about your server over UDP (compatible with Minecraft's implementation of the same). I'm thinking about taking on smelting next, and then I'll get back to improving the client renderer. Once the client renderer is done, like I've said in previous posts, I'll turn my focus towards lighting. I may take a few diversions for things like chest support in the meantime, though.
